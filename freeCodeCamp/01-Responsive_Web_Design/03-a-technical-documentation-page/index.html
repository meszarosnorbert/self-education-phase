Callback Functions

A callback függvény olyan függvény, amely egy másik függvény végrehajtása után kerül végrehajtásra. Általában egy másik függvény bemeneteként adják meg.

A callbackek megértése kritikus fontosságú, mivel a tömb metódusokban (például map(), filter()< stb.), setTimeout(), eventlistener-ekben (például click, scroll stb.) és még sok más helyen használják őket.

Íme egy példa egy " click " eventlistener-re egy callback függvénnyel, amely a gombra való kattintáskor lefut:

                    //HTML
                    <button class="btn">Click Me</button>

                    //JavaScript
                    const btn = document.querySelector('.btn');

                    btn.addEventListener('click', () => {
                    let name = 'John doe';
                    console.log(name.toUpperCase())
                    })
